# Транскрипты — Каракулина Валентина Ивановна

## 01 — Оригинальный транскрипт (AssemblyAI, март 2026)

**Файл:** `01_karakulina_original_assemblyai_20260326.txt`  
**Размер:** ~20 700 байт  
**Дата:** 26 марта 2026  
**Формат:** Никита диктует всё, что семья знает о Валентине  
**Использован в:** Stage1 (Fact Extractor → fact_map.json)

Это первый транскрипт проекта. Никита последовательно рассказывает биографию Валентины: детство, война, семья, переезды, работа в поликлинике. Включает текст рукописной автобиографии Валентины, воспоминания о характере, привычках, отношениях с семьёй. Именно на основе этого файла был построен `fact_map.json` в checkpoints.

## 02 — Уточняющее интервью Никиты с Татьяной (апрель 2026)

**Файл:** `02_karakulina_nikita_tatyana_interview.txt`  
**Размер:** ~51 700 байт  
**Дата:** 3 апреля 2026 (запись), очищено 12 апреля  
**Формат:** Диалог с разметкой спикеров (**Никита:** / **Татьяна:**)  
**Рассказчики:** Никита (внук, задаёт вопросы) + Татьяна (дочь, отвечает)  
**Использован в:** Phase B content_addition (run_20260412_phase_b)

Никита задавал уточняющие вопросы Татьяне по уже собранному материалу. Этот транскрипт добавил:
- историю с огурцами и чемоданом Маргося
- доктора Нинвана (Полсачева) как навязчивый авторитет
- продажу шубы ради пианино для Татьяны
- продажу дачи как главное сожаление
- традицию «посидеть на дорожку»
- «оловянный солдатик» — Татьяна не видела от неё слёз
- запах ванили и крема в доме
- детали конфликта с Маргосём (история со счётчиком)

**Важно:** история «шуба за пианино» есть в ОБОИХ транскриптах — Татьяна рассказала её дважды.
