## Контекст

По результатам проверки PDF Каракулиной (v10) есть три проблемы со стилем рендера, которые решаются в pdf_renderer.py и/или промпте Верстальщика.

## Что нужно сделать

### 1. Заполненность страниц — минимум 90%

Страницы не должны быть полупустыми. Минимальная заполненность — 90% высоты страницы текстом/блоками. Исключение: последняя страница главы (может быть заполнена меньше).

Если абзац не помещается на текущую страницу и оставляет >10% пустого места — подтянуть следующий абзац или перенести текущий на следующую страницу целиком.

### 2. Исторический блок — на всю ширину страницы

Образец: `historical_block_final.html` (приложен Дашей).

Спецификация:
- Фон: #f0ebe0 (тёплый бежевый)
- Левая черта: 2pt, #c4a070
- Лейбл: PT Sans Bold, 6.5pt, uppercase, #c4a070, letter-spacing 2.5px
- Текст: PT Serif Italic, 9pt, #5a4a38, line-height 14px
- Ширина: **полная ширина страницы** (без боковых полей, от края до края)
- Отступы: 10pt сверху/снизу, 14pt справа, 14pt слева (после черты)
- Декоративный год: убран
- Тёмный фон #1a1a2e: ЗАПРЕЩЁН

Сейчас исторический блок рендерится с боковыми полями — нужно сделать full-bleed (от края до края printable area).

### 3. Цитаты (callout) — по образцу из PDF v10

Текущая реализация близка, но нужно уточнить:
- Увеличенный интерлиньяж (больше чем у основного текста)
- Подпись: «— из воспоминаний семьи» — курсивом, PT Serif Italic, мелким кеглем, выравнивание по правому краю
- Только верхняя и нижняя линия #c4a070, без фона
- PT Serif Italic 13pt для текста цитаты
