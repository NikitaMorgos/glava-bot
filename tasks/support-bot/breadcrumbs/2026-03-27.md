# 2026-03-27 — Support Bot: создание и деплой

## Решения
- Модель: gpt-4o (по решению заказчика; gpt-4o-mini был бы дешевле, но gpt-4o качественнее)
- История диалога: in-memory dict (достаточно для MVP, теряется при рестарте)
- Групповой режим: ответ на @mention и reply, чтобы команда могла тестировать в общем чате
- System prompt строится динамически из KNOWLEDGE_BASE.md и TONE_OF_VOICE.md (кеш 5 мин)
- KB и ToV — отдельные markdown-файлы, чтобы команда могла редактировать без кода

## Контекст
- Задача создана по запросу заказчика
- Команда (Даша, Лена) тестирует в групповом чате Telegram
- Для группового чата нужно отключить privacy mode бота через @BotFather

## Деплой
- Файлы: `main.py`, `llm_support.py`, `support_prompt.py`, `AGENTS.md`, `tasks/support-bot/`
- Метод: scp (rsync недоступен на Windows)
- Сервер: 72.56.121.94, `systemctl restart glava`
- Статус: active (running), ошибок нет
- Smoke test: бот отвечает на вопросы в групповом чате через @mention
