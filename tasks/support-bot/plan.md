# Support Bot — План

## Цель
AI-бот клиентской поддержки в Telegram: отвечает на вопросы клиентов на основе базы знаний, работает в личных и групповых чатах.

### Фаза 2: Улучшения
- Сохранение истории поддержки в БД (для аналитики)
- Кнопка «Связаться с оператором» → уведомление команде
- Логирование вопросов без ответа (для расширения KB)
- Rate limiting (защита от спама)

### Фаза 3: Расширение
- Интеграция с admin-панелью (просмотр диалогов поддержки)
- Автоматическое расширение KB из новых вопросов
- Мультиязычность (если понадобится)

## Архитектура
```
Telegram (личный/группа)
  → main.py (handler)
    → llm_support.py
      → support_prompt.py (system prompt)
      → OpenAI API (gpt-4o)
    → ответ пользователю
```

## Конфигурация (.env)
```
OPENAI_API_KEY=...          # уже есть
SUPPORT_MODEL=gpt-4o        # модель для поддержки
```
