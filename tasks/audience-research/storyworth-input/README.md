# StoryWorth Input for CCO

Эта папка читается CCO в команде `/report storyworth` как дополнительный контекст.

## Что положено

- `rubricator.md` — категории из интерфейса StoryWorth.
- `questions_raw.txt` — сырой список вопросов (для дедупликации, кластеризации и оценки качества банка).

## Как запускать

В Telegram:

- `/report storyworth`
- или `@GLAVA_CCO_bot сделай отчёт по storyworth с учётом storyworth-input`
