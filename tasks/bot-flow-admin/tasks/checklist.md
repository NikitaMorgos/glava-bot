# Bot Flow в админке — чеклист

## Первый запуск

- [ ] Выполнить сид: `python scripts/seed_bot_messages.py`
- [ ] В `.env` бота: `ADMIN_API_BASE_URL=http://127.0.0.1:5001/api` (если бот и админка на разных серверах — указать URL админки)
- [ ] Запустить админку: `python admin/app.py` или через systemd
- [ ] Зайти в админку: https://admin.glava.family → Даша → Сообщения бота

## Редактирование сообщений

- [ ] Админка → Сообщения бота → выбрать сообщение → Редактировать
- [ ] Сохранить — изменения применяются сразу (бот кеширует 60 сек)

## Плейсхолдеры в шаблонах

- `{characters}` — список персонажей
- `{email}` — email пользователя
- `{total}` — сумма заказа
- `{payment_url}` — ссылка на оплату
