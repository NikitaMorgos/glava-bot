# Bot Flow в админке — статус задачи

| Поле | Значение |
|------|----------|
| **Задача** | Вывести пользовательский флоу бота в панель управления Даше |
| **Создана** | 2026-03-06 |
| **Статус** | 🟢 Готово |

---

## Решение (по аналогии с книгой)

| Компонент | Где | Кто меняет |
|-----------|-----|------------|
| **Тексты сообщений бота** | Админка → раздел «Сообщения бота» | Даша |
| **Сценарий бота** | Схема в админке (/dasha/bot_flow); n8n — опционально позже | Даша / разработчик |

---

## Прогресс

| Этап | Статус |
|------|--------|
| Хранилище (prompts, role=bot_*) | ✅ |
| API GET /api/prompts/bot_<key> | ✅ (существующий) |
| Раздел «Сообщения бота» в админке Даши | ✅ |
| Бот читает из API (fallback на prepay.messages) | ✅ |
| Схема сценария в админке | ✅ |
| n8n workflow Bot Flow | ⏳ Опционально |

---

## Запуск

1. Сид сообщений: `python scripts/seed_bot_messages.py`
2. Админка: https://admin.glava.family → Даша → Сообщения бота
3. Бот: `ADMIN_API_BASE_URL=http://127.0.0.1:5001/api` в .env (для доступа к сообщениям)
