## Живая карта флоу (добавлено 2026-03-17)

Страница `/dasha/bot_flow` — интерактивная карта всего пользовательского флоу бота:

- **Структура экранов и связей** — зашита в HTML (Entry → Интро → Персонажи → Email → Итог → Оплата → Блокировка → Онлайн-встреча + ошибки)
- **Тексты сообщений** — подтягиваются из БД через `GET /dasha/bot_messages_json` при открытии страницы
- **Кнопки** — реальные тексты из `prepay/keyboards.py`
- **Кнопка «↺ Обновить тексты»** — обновляет тексты без перезагрузки страницы
- **Клик по тексту** — разворачивает длинный текст
- **Ссылка «↗ редактировать»** — ведёт в редактор конкретного сообщения
- **Ссылка в сайдбаре** — «🗺 Карта флоу» в разделе продакта

### Файлы
- `admin/blueprints/dasha.py` — маршрут `/dasha/bot_flow` + эндпоинт `/dasha/bot_messages_json`
- `admin/templates/dasha/bot_flow.html` — шаблон живой карты (стиль по образцу Даши)

## Деплой (выполнен 2026-03-17)

1. `git push origin main` с локального компьютера
2. На сервере: `cd /opt/glava && git pull origin main`
3. `python scripts/seed_bot_messages.py` — сид 19 сообщений в БД
4. `sudo systemctl restart glava && sudo systemctl restart glava-admin`

### Известная проблема при деплое

Gunicorn (`glava-admin`) может зависнуть после перезапуска (воркер принимает TCP но не отвечает на HTTP).
Лечится: `sudo systemctl restart glava-admin`. После перезапуска работает нормально.
