## Версия 3.0 — текущая

### ✅ Завершено

**Структура и контент**
- [x] Единый файл `landing/index.html` — HTML/CSS/JS без зависимостей (кроме Google Fonts)
- [x] Hero — крупный заголовок «Глава», слоган, описание продукта, цена, два CTA, trust-маркеры
- [x] Блок ценностей — 3 карточки с заголовком секции и развёрнутыми описаниями
- [x] «Как это работает» — 3 шага, снижение тревоги, reassurance-note
- [x] Sample-блок — «Биография, а не расшифровка», 4 пункта качества, 2 CTA
- [x] Карусель отзывов — бесконечный цикл, random start при загрузке, swipe, dots, автопереключение 6.5 с
- [x] Mid-CTA полоса — после отзывов, тёмный фон, кнопка «Начать бесплатную консультацию»
- [x] FAQ — accordion, 6 вопросов в конверсионном порядке
- [x] Final CTA — тёмный блок, what-you-get из 3 пунктов, основная кнопка + подпись
- [x] Footer — навигация, соцсети-заглушки, контакты, копирайт, юридика

**Навигация**
- [x] Desktop-навигация (≥1025px): ссылки по секциям + кнопка «Начать» в шапке
- [x] Мобильная навигация: бургер → slide-in drawer справа, overlay, закрытие вне
- [x] Sticky nav — blur + shadow при скролле (`.scrolled` класс)

**Дизайн и адаптивность**
- [x] Container 1280px, breakpoint 1440px+ → 1360px
- [x] Базовый font-size: 17px, line-height 1.7
- [x] Breakpoints: 480px / 900px / 1025px (desktop nav) / 1440px
- [x] Палитра: cream / paper / parchment / gold (#a8823c) / ink (#181208)
- [x] Шрифты: Playfair Display (заголовки, italic) + Manrope (тело)

**FAQ — текущий порядок вопросов (конверсионный)**
1. Что именно я получу в итоге?
2. Что входит в 6 900 ₽ и нет ли скрытых доплат?
3. Сколько времени занимает весь процесс?
4. Что если близкий немногословен?
5. Можно работать без голосовых — только в переписке?
6. Как вы обращаетесь с личными данными?
