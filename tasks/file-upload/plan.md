# File Upload — План

## Проблема
Telegram Bot API ограничивает скачивание файлов ботом до 20 МБ.
Часовое интервью в mp3 (128 kbps) = ~55 МБ — бот не может его скачать.
Видеофайлы бот ранее игнорировал полностью.

## Решение
Веб-страница загрузки файлов (`cabinet.glava.family/upload/<token>`).
Бот автоматически определяет, когда файл нельзя обработать через Telegram, и даёт одноразовую ссылку на загрузку через браузер.

## Архитектура

```
Клиент отправляет файл > 20 МБ / видео в Telegram
  → Бот проверяет file_size или тип (видео)
    → "Файл нельзя обработать через Telegram" + кнопка [Загрузить через браузер]
      → Кнопка ведёт на cabinet.glava.family/upload/<token>
        → Веб-страница: drag & drop / выбор файла
          → POST /upload/<token> → multipart upload
            → Сервер: сохраняет в S3, записывает в БД
              → Бот отправляет уведомление клиенту в Telegram
                → Запускается пайплайн транскрипции

Клиент вводит /upload
  → Бот генерирует токен и даёт ссылку напрямую (без необходимости отправлять файл)
```

## Безопасность
- Одноразовый токен (UUID) с TTL 1 час
- Токен привязан к telegram_id
- После использования токен удаляется
- Лимит размера файла: 500 МБ (на уровне nginx + Flask)
- Только аудиофайлы (mp3, ogg, m4a, wav, opus, oga)
