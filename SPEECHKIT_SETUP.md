# Настройка Yandex SpeechKit для транскрибации

## Шаг 1. Создать API-ключ

1. Зайди в [Yandex Cloud Console](https://console.cloud.yandex.cloud)
2. **Сервисные аккаунты** → выбери `glava-bot`
3. Нажми **Создать новый ключ** → **Создать API-ключ**
4. Скопируй ключ (показывается один раз)

## Шаг 2. Добавить в .env

```
YANDEX_API_KEY=твой_api_ключ
```

## Шаг 3. Роль для SpeechKit

Сервисный аккаунт `glava-bot` уже имеет `storage.editor` — этого достаточно для чтения файлов из бакета. SpeechKit будет получать аудио по временной ссылке.

## Готово

После этого при экспорте (`export_client.py TELEGRAM_ID`) голосовые будут транскрибироваться через SpeechKit.
