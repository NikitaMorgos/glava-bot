01 — Проблема и решение

Почему семейные истории теряются

Без GLAVA
  • 😓Голосовые истории хранятся в мессенджерах и теряются
  • 📁Тысячи фото без подписей и систематизации
  • ⏳Написать биографию с нуля — долго, дорого, непонятно с чего начать
  • 🏢Профессиональные издательства — дорогостоящий и долгий процесс
  • 💔Старшее поколение не пользуется сложными приложениями
VS
С GLAVA
  • 📱Привычный Telegram — никакого нового приложения
  • 🎙Голосовые сообщения → автоматическая транскрипция с диаризацией
  • 🤖10+ AI-агентов редактируют, проверяют факты, верстают книгу
  • 📖Готовый PDF с обложкой — за несколько часов
  • 💳Оплата онлайн, доставка прямо в чат

06 — AI-агенты

Команда из 10+ AI-специалистов

🗺
Fact Extractor
Извлекает имена, даты, места из транскрипта. Строит структурированную факт-карту.
→ fact_map.json
📚
Историк
Контекстуализирует события, добавляет историческую справку эпохи.
→ context_layer
✍️
Ghostwriter
Пишет биографический нарратив по фактам и контексту. Центральный агент пайплайна.
→ bio_draft.md
✅
Fact Checker
Сверяет черновик с факт-картой. Возвращает ghostwriter на доработку при несоответствиях.
→ pass / iterate
✏️
Литературный редактор
Улучшает стиль, ритм, образность. Правит нарратив без искажения фактов.
→ edited_draft
🔍
Корректор
Исправляет орфографию, пунктуацию, грамматику. Финальный текстовый фильтр.
→ clean_text
🎨
Cover Designer
Генерирует концепцию обложки. Запускает Replicate для создания визуала.
→ cover.png
📐
Layout Designer
Верстает PDF: типографика, расстановка фото, стили. QA-цикл по WCAG.
→ layout.pdf
❓
Interview Architect
Генерирует персонализированные вопросы для углублённого интервью.
→ questions.json
📦
Продюсер
Финальная упаковка: объединяет PDF, фото, вопросы. Отправляет через Telegram API.
→ deliver 🚀
🔀
Triage
Маршрутизатор: определяет тип входящего запроса и направляет в нужную ветку пайплайна.
→ routing
💬
Support Bot (CCO)
GPT-4o агент поддержки. Отвечает на вопросы клиентов, работает в отдельном боте.
→ ответ клиенту

Конкурентные преимущества

🔒

Paywall-first архитектура

Пайплайн, транскрипция и обработка файлов запускаются только после подтверждения оплаты. Защита от злоупотреблений.

🔁

QA-циклы с обратной связью

Агенты итерируют до получения качественного результата — автоматическая проверка фактов, стиля, вёрстки.

🧩

Версионирование промптов

Операторы меняют поведение агентов через Admin API без деплоя кода. Быстрые итерации продукта.

📡

Мультиканальный сбор данных

Голос, фото, веб-загрузка >20 МБ, запись онлайн-встреч (Zoom, Telemost, Jitsi) — всё в одной системе.
