### Конфигурация в `pipeline_config.json`

```json
{
  "agent_name": {
    "model": "claude-sonnet-4-20250514",
    "max_tokens": 8000,
    "temperature": 0.2,
    "prompt_file": "XX_agent_name_v1.md"
  }
}
```

**Почему такие параметры:**
- `temperature`: низкая → точность / высокая → вариативность
- `max_tokens`: объяснение почему именно столько
