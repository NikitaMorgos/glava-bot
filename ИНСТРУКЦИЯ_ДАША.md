# Инструкция для Даши — экспорт данных клиента

## Что это

Скрипт собирает все данные одного клиента (голосовые + фото с подписями) в **одну папку**. Готово для верстки книги.

---

## Шаг 1. Открыть проект

Перейди в папку проекта GLAVA и открой терминал (PowerShell).

---

## Шаг 2. Посмотреть список клиентов

```powershell
.\venv\Scripts\python.exe export_client.py
```

Появится список:
```
Клиенты (telegram_id — голосовых — фото с подписями):

  123456789  @username  — 5 голос., 3 фото

Экспорт: python export_client.py TELEGRAM_ID
```

Запомни **telegram_id** нужного клиента (число слева).

---

## Шаг 3. Экспортировать клиента

```powershell
.\venv\Scripts\python.exe export_client.py 123456789
```

(подставь свой telegram_id вместо 123456789)

---

## Шаг 4. Где искать результат

Папка создаётся в:
```
GLAVA/exports/client_123456789_username/
```

Внутри:
- **voice/** — голосовые (001.ogg, 002.ogg...)
- **photos/** — фото (001.jpg, 002.jpg...)
- **transcript.txt** — текст интервью (транскрипция голосовых)
- **captions.txt** — подписи к фото
- **manifest.txt** — полное описание содержимого

---

## Итого

1. `.\venv\Scripts\python.exe export_client.py` — список клиентов
2. `.\venv\Scripts\python.exe export_client.py TELEGRAM_ID` — экспорт
3. Папка в `exports/` — всё для верстки (включая transcript.txt)

**Примечание:** для транскрибации нужен ffmpeg. Скачай с ffmpeg.org и добавь в PATH.
