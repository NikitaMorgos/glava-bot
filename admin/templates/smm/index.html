Как работает SMM-редакция
Полная последовательность от материала до публикации
1
🔍 Скаут собирает материал и генерирует темы

Берёт: промпт Стратегии + промпт Скаута + промпт Площадки + все активные Рубрики + отзывы клиентов из БД + твои ручные идеи. Отдаёт Claude → получает список тем с рубриками.

2
✍️ Журналист пишет статью

Берёт тему → пишет через GPT-4o в стиле, заданном промптом Журналиста (твой стиль: Гришковец, Минаев, личные интонации). Без жёсткого шаблона структуры — всё управляется промптом.

3
📋 Редактор проверяет текст

Claude с промптом Редактора: одобряет если статья по теме и без грубых ошибок, отклоняет только явно неподходящий контент. Оставляет комментарий с рекомендациями.

4
🎨 Иллюстратор создаёт обложку

Claude с промптом Иллюстратора читает текст статьи → формулирует визуальный концепт (image prompt на английском, стиль nana-banana) → Replicate генерирует картинку.

5
👁 Ты проверяешь и одобряешь

Открываешь пост, читаешь текст, смотришь картинку. Можно отредактировать текст руками, перегенерировать обложку с другим промптом. Нажимаешь «Одобрить».

6
🚀 Публикация на площадке

Playwright автоматически открывает Яндекс Дзен, вставляет заголовок, текст, загружает обложку, нажимает «Опубликовать». Требует сохранённой сессии (один раз).

Что ты настраиваешь в Настройках SMM
📂 Рубрики — направления контента. Например: «Отзывы клиентов», «Бэкстейдж», «Истории знаменитостей». Скаут распределяет темы по активным рубрикам.
🌐 Площадки — каналы публикации. Промпт площадки задаёт формат, длину, тональность конкретно для Дзена, ВКонтакте и т.д.
🎭 Роли — промпты всех участников редакции: Скаут, Журналист (стиль текста), Редактор (критерии проверки), Иллюстратор (стиль визуала, nana-banana).
Статусы: Черновик → Генерируется → Текст готов → Готов к одобрению → Одобрен → Опубликован · Если редактор отклонил — отредактируй руками и одобри вручную.
