Как работает SMM-редакция v2
Полная цепочка
1
🔍 Скаут генерирует темы

Для каждой темы указывает рубрику и площадку/формат.

system: Скаут Стратегия + все Площ./Форм. + все Рубрики
2
✍️ Журналист (автоматический подбор) пишет статью

По рубрике + формату выбирается журналист с нужными назначениями. Получает промпт рубрики + описание формата.

system: промпт журналиста + промпт рубрики + описание формата
3
🎨 Иллюстратор → обложка
system: Иллюстратор заголовок + фрагмент текста
4
📋 Выпускающий редактор → решение
system: Редактор текст + image prompt
5
👁 Лена одобряет → 📅 дата → 🚀 публикация

Назначь дату, одобри — и пост уйдёт на площадку.

Настройки: Площадки/Форматы (~20 связок) · Рубрики · Журналисты (с назначениями) · Служебные роли
